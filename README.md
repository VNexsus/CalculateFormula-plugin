## Плагин пошагового расчета формул в табличном редакторе Р7-Офис
Плагин позволяет производить пошаговый расчет результирующих значений в ячейках, содержащих формулы.

<img alt="Preview" width="590px" src="https://github.com/VNexsus/CalculateFormula-plugin/blob/main/Preview.png">


## Функциональные возможности:
Выбор ячейки с формулой, просмотр содержащейся формулы, цветовая подсветка зависимых ячеек на листе, пошаговое вычисление результата.

## Применимо для продуктов:
*	Десктоп-редакторы Р7-Офиc 2025.3.1.923 и выше

## Как установить
*	Для установки плагина скачайте файл <a href="https://github.com/VNexsus/CalculateFormula-plugin/blob/main/CalculateFormula.plugin">CalculateFormula.plugin</a> следуйте инструкциям на сайте <a href="https://support.r7-office.ru/desktop_editors/plugins-and-macros/plugins-plugins-and-macros/adding-plugins/p7-desktop-editors/">Р7-Офиc<a/>
